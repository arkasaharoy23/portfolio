# Arka Saha Roy — Personal Portfolio
 
A modern, fully responsive personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks.
 
## Tech Stack
- HTML5 (Semantic)
- CSS3 (Custom Properties, Grid, Flexbox, Glassmorphism)
- JavaScript (ES6+, Intersection Observer API)

## Features
- Dark theme with blue/purple gradient accents
- Glassmorphism design language
- Smooth scroll navigation with active link highlighting
- Scroll-triggered reveal animations (Intersection Observer)
- Animated skill bars
- Mobile-first responsive design
- Hamburger nav menu for mobile
- Loading screen animation
- Back-to-top button
- Animated hero background (floating orbs + grid)
- Responsive across all devices
